### 方法2: 测试Agent连接（需要终端）

## 🐛 遇到问题？

### 问题：npm install 失败
**解决**：
```bash
# 清除缓存重新安装
rm -rf node_modules package-lock.json
npm install
```

### 问题：端口被占用
**解决**：
```bash
# 查找占用进程
lsof -i :5173

# 杀死进程
kill -9 <PID>

# 或者使用其他端口
npm run dev -- --port 3000
```

### 问题：Agent连接失败
**解决**：
1. 确认Agent正在运行
2. 检查端口8081：`lsof -i :8081`
3. 检查防火墙设置
4. 查看Agent日志排查错误

### 问题：页面显示空白
**解决**：
1. 打开浏览器控制台（F12）查看错误
2. 确认所有依赖已安装
3. 清除浏览器缓存
4. 重启开发服务器
